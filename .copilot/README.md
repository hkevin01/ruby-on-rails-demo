# Copilot Configuration

This directory contains GitHub Copilot configuration files for the Ruby on Rails Blog Application.

## Files

### copilot-instructions.md
Contains project-specific instructions for GitHub Copilot to provide better context-aware suggestions.

### copilot-ignore.txt
Specifies files and patterns that should be ignored by GitHub Copilot when providing suggestions.

## Usage

GitHub Copilot will automatically use these configuration files when working in this project to provide more relevant and context-aware code suggestions.

For more information about GitHub Copilot configuration, see the [official documentation](https://docs.github.com/en/copilot).
